# Инструкция по переносу всех тренировок

1. Откройте файл src/screens/WorkoutsScreen.tsx
2. Найдите объявление массива:
   const workouts = [
   ...
   ];
3. Скопируйте весь массив (от [ до ] и ;) — все объекты с id: 1 по id: 54.
4. Откройте файл src/data/workouts.ts
5. Замените содержимое массива workouts на скопированный массив из предыдущего шага:
   export const workouts = [ ...сюда... ];
6. Убедитесь, что в src/screens/WorkoutsScreen.tsx и src/App.tsx используется импорт:
   import { workouts } from '../data/workouts';
7. Перезапустите приложение и проверьте, что отображаются все 54 тренировки и прогресс-бар работает корректно.

Если потребуется — повторите шаги или обратитесь к этому файлу для контроля.
